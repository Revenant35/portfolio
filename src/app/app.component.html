Project Showcase

Real-time Pose Estimation Pipeline

I constructed my own realtime pose estimation pipeline using CV fundamentals and Tensorflow. The pipeline was able to detect and track human poses in real-time using a webcam feed. The pipeline was built using a custom-trained PoseNet model and was able to detect multiple poses simultaneously. The resulting poses were then rendered on the webcam feed in real-time.

Medication Logging Application

I developed a medication logging iOS application called TapRx. TapRx was designed to help users keep track of their medications and when they should take them. The application featured a medication list, a schedule, and a dashboard to help users visualize their medication intake. The frontend was built using Swift and SwiftUI, while the backend was built using Flask and Firebase.

Illumica

As a founder of Illumica, I have been working on a healthcare analytics platform that helps healthcare providers visualize and analyze their patient data. The platform features a dashboard, a pose estimation pipeline, and analytics tools to help healthcare providers make informed decisions about their patients. The platform is built using Angular, Node.js, and MariaDB.
